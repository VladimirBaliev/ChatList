# ChatList

Программа для отправки одного промта в несколько нейросетей и сравнения их ответов.

## Требования

- Python 3.11 или выше
- Установленные зависимости (см. requirements.txt)

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` в корне проекта с вашими API-ключами:
```
OPENAI_API_KEY=your_openai_api_key_here
DEEPSEEK_API_KEY=your_deepseek_api_key_here
GROQ_API_KEY=your_groq_api_key_here
```

## Запуск

### Способ 1: Через командную строку
```bash
python main.py
```

### Способ 2: Через скрипт (Windows)
Дважды кликните на `run.bat` или выполните в PowerShell:
```powershell
.\run.ps1
```

### Способ 3: Через скрипт (PowerShell)
```powershell
python main.py
```

**Важно:** Убедитесь, что вы запускаете именно `main.py`, а не другие файлы!

## Использование

1. Введите промт в текстовое поле или выберите сохраненный промт
2. Выберите модели, в которые хотите отправить запрос (отметьте чекбоксы)
3. Нажмите кнопку "Отправить"
4. Дождитесь получения ответов от всех моделей
5. Отметьте результаты, которые хотите сохранить
6. Нажмите "Сохранить выбранные" для сохранения в базу данных

## Структура проекта

- `main.py` - главный модуль с интерфейсом
- `db.py` - модуль для работы с базой данных
- `models.py` - модуль для управления моделями нейросетей
- `network.py` - модуль для отправки запросов к API
- `chatlist.db` - база данных SQLite (создается автоматически)


а не другие файлы.
